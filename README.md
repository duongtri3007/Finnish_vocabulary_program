# WordNest
Learning Finnish every day

## 🌟 Highlights
- Focusing on learning vocabulary.
- Having flashcard and multiple choices practices.
- Following Django web framework to develop the project.
- Using Django session to create temporary sessions for users (Without database).

## Overview
- We are Vietnamese team who want to learn Finnish when studying at SAMK, Pori, Finland.
- Our purpose is to create and deploy a website to practice with learned programming skils.

## Technologies
- HTML
- CSS
- JavaScript
- Python
- API
- FastAPI
- Django

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/duongtri3007/Finnish_vocabulary_program.git
```

Create a virutal environemnt:
```bash
python -m venv myenv
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the application with:

```bash
python3 manage.py runserver
```
